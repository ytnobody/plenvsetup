# plenvsetup

plenv setup automation tool 

## SYNOPSIS

On bash,

    $ curl -L http://is.gd/plenvsetup | bash
    $ . ~/.bash_profile

Or, on zsh,

    $ curl -L http://is.gd/plenvsetup | zsh
    $ . ~/.zprofile

## ENVIRONMENT

- bash
- zsh

## REQUIRES

- perl
- curl
